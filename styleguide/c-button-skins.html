Buttons (skins)

This page documents the different button skins.

Button primary

# 
<button class="c-button c-button--primary" type="button">
    <span class="c-button__content">
        <span class="c-button__label">Button label</span>
    </span>
</button>
<button class="c-button c-button--primary" type="button" disabled="disabled">
    <span class="c-button__content">
        <span class="c-button__label">Button label</span>
    </span>
</button>
+c-button({skin: 'primary'}) Button label
+c-button({skin: 'primary'})(disabled) Button label

Possible parameters for buttons

Parameters Type Possible values Description
buttonType string button, submit, reset The default value of a button type is button.
href string This can be any url or id. If this parameter is used then this button will automatically become a link.
icon string See icon style guide for available icons. This parameter is used for buttons with icons.
skin string primary, secondary, borderless, muted Determines how the button is visually rendered.
layout string text-icon, icon-only, default Determines the placement of the icon. The default layout is icon-text.
isBlock boolean true, false Determines if the button is a full width block button. The default state is false
isLink boolean true, false Determines if the button is a link button. The default state is false
