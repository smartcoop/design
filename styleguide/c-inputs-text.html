Inputs (text)

This page contains examples of typical form elements. For examples of full forms, please refer to the form groups object.

Text input

#

A standard text input. Use for single line text. Avoid using the placeholder attribute, unless really necessary.

<div class="o-form-group-layout o-form-group-layout--standard">
    <div class="o-form-group">
        <label class="o-form-group__label" for="textFieldId">Label</label>
        <div class="o-form-group__controls">
            <input class="c-input" type="text" value="Default Input" id="textFieldId" />
        </div>
    </div>
    <div class="o-form-group">
        <label class="o-form-group__label" for="textFieldId">Label</label>
        <div class="o-form-group__controls">
            <input class="c-input" type="text" disabled="disabled" value="Default Input" id="textFieldId" />
        </div>
    </div>
</div>

    .o-form-group
        label.o-form-group__label(for="textFieldId") Label
        .o-form-group__controls
            input.c-input(type="text" value="Default Input" id="textFieldId")
    .o-form-group
        label.o-form-group__label(for="textFieldId") Label
        .o-form-group__controls
            input.c-input(type="text" disabled value="Default Input" id="textFieldId")

Textarea

#

A standard textarea. Use for multi-line text. Manipulate the height with the rows attribute or the following modifier classes: .c-input--h-small, .c-input--h-medium, .c-input--h-large.

Optionally add a class of .autosize to autosize. This functionality relies on autosize.

<div class="o-form-group-layout o-form-group-layout--standard">
    <div class="o-form-group">
        <label class="o-form-group__label" for="textareaId">Label</label>
        <div class="o-form-group__controls">
            <textarea class="c-textarea" rows="5" id="textareaId"></textarea>
        </div>
    </div>
    <div class="o-form-group">
        <label class="o-form-group__label" for="textareaId2">Label</label>
        <div class="o-form-group__controls">
            <textarea class="c-textarea" rows="5" disabled="disabled" id="textareaId2"></textarea>
        </div>
    </div>
</div>
.o-form-group-layout.o-form-group-layout--standard
    .o-form-group
        label.o-form-group__label(for="textareaId") Label
        .o-form-group__controls
            textarea.c-textarea(rows="5" id="textareaId")
    .o-form-group
        label.o-form-group__label(for="textareaId2") Label
        .o-form-group__controls
            textarea.c-textarea(rows="5" disabled id="textareaId2")
