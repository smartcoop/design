Markdown Data

You can use markdown in templates by using the marked filter.

As an example, make put your markdown in a variable.

block append pageVariables
    - var myContent = '### I am a 3rd level heading'

Then render it with the following code:

div!= marked(myContent)

Example output:

I am a 3rd level heading
