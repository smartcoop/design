Possible parameters for badges

Parameters Type Possible values Description
skin string default, help, error, warning, success Determines the skin of the badge.
icon string See icon style guide for available icons. This parameter is used for the icon. If an icon is not set, a fallback icon is rendered.
size string small Determines the size of the badge
