Avatar link

<a class="c-avatar c-avatar--img" href="/">
    <img src="http://placekitten.com/500/500" alt="avatar" />
</a>
+c-avatar({href: '/', img: 'http://placekitten.com/500/500', alt: 'avatar'})

Possible parameters for avatars

Parameters Type Possible values Description
img string Determines if an image is set to an avatar.
alt string Determines if an alt tag is set to an avatar.
icon string See icon style guide for available icons. This parameter is used for the icon. If an icon is not set, a fallback icon is rendered.
size string small Determines the size of the avatar
href string This can be any url or id. If this parameter is used then this avatar will automatically become a link.
